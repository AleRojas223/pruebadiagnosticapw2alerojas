Ale Rojas Cuetionario

1. ¿Qué es HTML?
HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Define la estructura y el contenido de una página web mediante el uso de etiquetas.
2. ¿Cómo se escriben las instrucciones en HTML?
Las instrucciones en HTML se escriben mediante etiquetas, que son palabras clave rodeadas por corchetes angulares (< >). La mayoría de las etiquetas tienen una de apertura (ej. <p>) y una de cierre (ej. </p>), con el contenido entre ellas.

6. ¿Qué es CSS?
CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento HTML. Controla aspectos como colores, fuentes, márgenes, diseños y otros elementos visuales.
7. ¿Cómo puedo vincular una hoja de estilo a una página Web?
Puedes vincular una hoja de estilo CSS a una página HTML usando la etiqueta <link> en la sección <head> del documento HTML. Ejemplo:

<link rel="stylesheet" href="estilos.css">

9. ¿Qué es Javascript?
JavaScript es un lenguaje de programación interpretado que se ejecuta en el navegador del cliente. Permite crear páginas web interactivas, manipular el DOM, realizar peticiones asíncronas y muchas otras funcionalidades dinámicas.
11. ¿Cuáles son los métodos de envío de un formulario?
Los métodos principales son:
  • GET: Los datos del formulario se envían como parámetros en la URL. Tiene límite de tamaño y es visible en la barra de direcciones.
  • POST: Los datos se envían en el cuerpo de la petición HTTP. No tiene límite de tamaño y no es visible en la URL.
Se especifican en el atributo "method" de la etiqueta <form>.
12. ¿Qué es PHP?
PHP (Hypertext Preprocessor) es un lenguaje de programación del lado del servidor ampliamente utilizado para el desarrollo web. Se incrusta en HTML y se ejecuta en el servidor antes de que la página sea enviada al navegador del cliente. Es especialmente útil para interactuar con bases de datos y crear contenido dinámico.
